# Skills — Criação de Sites

Subset neutro extraído do repo central-do-frete (sem dependências de domínio).

## Instalação

Copiar para `.claude/skills/` do repo novo (project) ou `~/.claude/skills/` (global).

## Conteúdo

| Skill | O quê |
|---|---|
| `ui-ux-pro-max/` | UI/UX intelligence — 50+ styles, 161 palettes, 57 font pairings, UX guidelines, charts |
| `tailwind-design-system/` | Design systems com Tailwind v4 — tokens, component libraries, responsive patterns |
| `benchmark-visual/` | Benchmarking competitivo visual (neutralizado — sem Clarity/domínio) |
| `find-skills/` | Descoberta e instalação de novas skills |
| `terse.md` | Modo comunicação mínima (mover para `.claude/rules/` se quiser sempre ativo) |
| `playwriter-collaborative.md` | Padrão de browser automation colaborativa (requer Playwriter MCP) |

## Já cobertas por skills globais (não incluídas)

impeccable, frontend-design, ui-animation, design-motion-principles, human-image — vivem em `~/.claude/skills/` e plugins, valem em qualquer repo.
