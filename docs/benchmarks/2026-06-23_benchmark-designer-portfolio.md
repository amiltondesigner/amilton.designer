# Benchmark Visual — Portfólio de Designer/Freelancer

**Data:** 2026-06-23
**Tag:** [P1]
**Status:** Concluído
**Feature:** Hero, grid de projetos, case study (detalhe) e CTA para portfólio de UI/UX designer freelancer.

---

## Resumo Executivo

O mercado de portfólios de designer em 2026 abandonou o template "centralizado e seguro" em favor de composições **expressivas e editoriais**: heros type-first com tipografia grande, grids assimétricos/"quebrados", e uso confiante de preto-e-branco com **um único acento de cor** — exatamente a direção definida (monocromático grafite + azul cirúrgico). Reviewers **escaneiam, não leem linearmente**: decidem em segundos se o trabalho é relevante e se o raciocínio demonstra maturidade. O portfólio atual do Amilton é coeso mas "template"; as maiores oportunidades são tipografia expressiva, variedade visual no grid, e páginas de detalhe estruturadas como case study real (problema → processo → resultado).

---

## Patterns Recorrentes (convergência de mercado)

| Pattern | Relevância | Aplicação aqui |
|---|---|---|
| **Type-first hero** (tipografia grande, expressiva, assimétrica) | Alta | Hero da home: aumentar escala do título, menos "card", mais editorial |
| **Especialização óbvia em segundos** | Alta | Deixar claro "UI/UX Designer" + nicho/valor no topo |
| **Grid com variedade > cronologia** ("broken grid") | Alta | Cards de projeto com tamanhos/ênfases variados |
| **About preview com foto profissional** | Média | Já existe — manter, integrar melhor ao hero |
| **B&W + 1 acento de cor** | Alta | Valida monocromático + azul |
| **Micro-interações / transições fluidas** | Alta | Hover nos cards, reveal on-scroll suave (150–300ms) |
| **Case study: Overview → Processo → Resultado** | Alta | Reestruturar template de detalhe |
| **Navbar fixa + prev/next no fim do case** | Alta | Adicionar navegação entre projetos |
| **Desenhar para escaneabilidade** | Alta | Hierarquia forte, eyebrows, espaçamento generoso |

## Produto Atual vs Mercado

| Aspecto | Hoje | Melhor do Mercado | Gap |
|---|---|---|---|
| Hero | Card centralizado, título médio | Type-first, escala grande, editorial | Médio-Alto |
| Tipografia | Inter (genérica) | Pares com caráter (designer) | Alto |
| Grid de projetos | Uniforme 2×2 | Variedade/ênfase, hover rico | Médio |
| Página de detalhe | Boa, mas linear | Case study escaneável + prev/next | Médio |
| Cor | Azul espalhado | Acento cirúrgico | Médio |
| Motion | Estático | Micro-interações fluidas | Médio |

## Oportunidades Priorizadas (Impacto × Esforço)

1. **Tipografia expressiva** (Archivo + Space Grotesk) — Impacto Alto, Esforço Médio. Quick win de percepção.
2. **Acento azul cirúrgico + base monocromática** — Impacto Alto, Esforço Baixo.
3. **Hero type-first** — Impacto Alto, Esforço Médio.
4. **Detalhe como case study + prev/next** — Impacto Alto, Esforço Médio-Alto.
5. **Micro-interações nos cards** — Impacto Médio, Esforço Baixo.
6. **Grid com variedade** — Impacto Médio, Esforço Médio.
